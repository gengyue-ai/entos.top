## 一、复盘

### 1.1 发生了什么

| 阶段 | 时长 | 结果 |
|------|:----:|:----:|
| Blender API 探索 | ~4h | ❌ Blender 5.1.2 FFMPEG 编译不完整 |
| 隧道 + 页面架构 | ~2h | 🟡 部分可用 |
| 本地 Agent 开发 | ~2h | ❌ 未完成 |
| FFmpeg 混剪测试 | ~3h | ✅ 基础拼接已出片 |

### 1.2 核心问题

- **跳步：** 没先确认核心链路能不能跑，直接写三层代码
- **猜测代替验证：** Blender 5.1.2 API 全凭猜
- **让你反复做体力活：** 删文件→重下载→重运行
- **脱离实际需求：** 想要政府项目品质宣传片，我在纠结 MP4 输出

### 1.3 关键发现

- Blender 5.1.2：FFMPEG 输出未完全编译
- FFmpeg 8.1.2：完整支持 H.264、AAC、转场、字幕、混音
- 架构方向正确：SaaS 页面 + 云端调度 + 本地 Agent + 本地渲染

## 二、系统架构

```
用户浏览器          阿里云服务器          你的电脑
    │                    │                    │
    │  SaaS 页面         │  FastAPI 调度      │  Agent
    │  ┌──────────┐     │  ┌──────────┐     │  ┌────────┐
    │  │ 素材配置  │────→│  │ 任务接收  │────→│  │FFmpeg  │
    │  │ 参数设置  │     │  │ 进度跟踪  │     │  │素材分析 │
    │  │ 进度监控  │←────│  │ 历史记录  │←────│  │输出文件 │
    │  │ 任务管理  │     │  └──────────┘     │  └────────┘
    │  └──────────┘     └────────────────────┘         │
    │                           │                     │
    │                    WebSocket 隧道                │
    └──────────────────────────────────────────────────┘
```

| 组件 | 技术 | 位置 | 状态 |
|------|------|------|:----:|
| SaaS 页面 | HTML + JS | entos.top/clipforge/ | ✅ 已有 |
| FastAPI 后端 | Python | 阿里云 :8765 | ✅ 已有 |
| WebSocket 隧道 | websockets | 阿里云:8768 ↔ 本地 | ✅ 已有 |
| 本地 Agent | Python | 你的电脑 | 🟡 待完善 |
| FFmpeg | 8.1.2 | C:\\Windows\\ffmpeg.exe | ✅ 已有 |

| 参数 | 说明 | 默认值 |

## 四、前后端方案

### SaaS 页面（已有原型，需完善）

`entos.top/clipforge/` - 深色主题，含仪表盘、配置面板、模板选择、进度条、日志、历史记录。

### 云端 API（已有，需完善）

| 路由 | 方法 | 说明 |
|------|:----:|------|
| /api/agent/status | GET | Agent 在线状态 |
| /api/agent/task | POST | 创建任务（含全部参数） |
| /api/agent/progress/{id} | GET | 任务进度 |
| /api/agent/history | GET | 历史任务 |
| /api/agent/cancel/{id} | POST | 取消任务 |

### 隧道（已有）
WebSocket 隧道（阿里云:8768 ↔ 本地），任务通过数据库轮询转发。

### 本地 Agent（需重写）

| 功能 | 实现 | 优先级 |
|------|------|:------:|
| 接收任务 | WebSocket 接收参数 | P0 |
| 素材分析 | FFmpeg 场景检测 + 运动分析 | P1 |
| 智能编排 | 按精彩度排序 + 节奏控制 | P1 |
| FFmpeg 渲染 | 截取 → concat → 转场 → 字幕 → 配乐 | P0 |
| 进度上报 | WebSocket 实时推送 | P0 |
| Blender 渲染 | 特殊特效场景 | P2 |
