# Remote PC Agent Windows 客户端 — 产品方案

## 一、产品定位

一套即装即用的本地电脑客户端。运营人员安装后，授权云端 AI 远程操控本地电脑执行任务（混剪视频、批量处理文件、运行脚本等）。

**核心价值：运营人员不需要懂技术，安装即用。**

## 二、目标用户

- 运营人员：负责素材管理和任务发起，不写代码
- 企业管理者：购买服务，分配给团队成员使用
- AI Agent：云端智能体，通过客户端操控本地资源

## 三、安装体验

### 3.1 获取安装包

| 渠道 | 说明 |
|------|------|
| 官网下载 | https://entos.top/download/ |
| 管理员分发 | 企业批量部署时通过内部渠道分发 |

### 3.2 安装流程

```
下载 RemotePcAgent_Setup.exe（约 50MB）
  ↓
双击运行 → 选择安装目录 → 下一步
  ↓
输入授权码（企业管理员提供）
  ↓
完成安装 → 自动启动主界面
```

**安装包包含：**
- Python 运行时（嵌入式）
- MCP Server（HTTP/SSE）
- WebSocket 隧道客户端（自动连接云端）
- PyInstaller 打包为单 exe

### 3.3 首次启动

```
启动 → 显示授权码输入框
  ↓
输入授权码 → 验证通过
  ↓
自动检测本地环境（FFmpeg / Blender / 磁盘空间）
  ↓
进入主界面 → 绿色状态灯亮起 → 已就绪
```

## 五、任务模板

### 5.1 预置模板

| 模板名称 | 适用场景 | 输入 | 输出 |
|---------|---------|------|------|
| 足球赛事宣传片 | 体育赛事混剪 | 花絮素材 | 1-2分钟宣传片 |
| 探店短视频 | 商业探店 | 店内拍摄素材 | 30-60秒成片 |
| 活动回顾 | 会议/活动 | 全程录像 | 3分钟精华集锦 |
| 产品展示 | 电商/展销 | 产品特写素材 | 30秒展示片 |
| 自定义 | 通用 | 任意素材 | 按需定制 |

### 5.2 模板可配置参数

每个模板后台配置：
- 目标时长（秒）
- 音乐风格（激昂/温馨/科技感）
- 字幕风格（大标题/底部字幕/无）
- 片头/片尾模板
- 转场类型

**配置在云端维护，客户端启动时自动同步最新模板。运营人员不需要改配置。**

## 六、技术架构

### 6.1 客户端内部结构

```
RemotePcAgent.exe
├── MCP Server（HTTP/SSE，端口 8767）
│   ├── run_shell        — 执行本地命令
│   ├── run_blender      — 后台渲染
│   ├── scan_files       — 扫描素材
│   ├── read/write_file  — 读写文件
│   └── get_system_info  — 系统状态
├── Tunnel Client（WebSocket → 阿里云）
│   └── 自动连接 ws://47.115.50.15:8768
├── GUI 服务（HTTP 本地页面）
│   └── 自动打开浏览器 http://localhost:18767
└── 自动更新模块
    └── 启动时检查新版本
```

### 6.2 通信流程

```
运营人员点"开始任务"
  ↓
客户端发送任务请求到云端
  ↓
云端 Hermes 分析素材 → 写 Blender/FFmpeg 脚本
  ↓
通过隧道调用客户端 MCP 工具执行
  ↓
客户端本地渲染 → 进度回传
  ↓
输出文件到指定目录 → 通知运营人员
```

### 6.3 技术选型

| 组件 | 选型 | 理由 |
|------|------|------|
| 打包 | PyInstaller | 单 exe，包含 Python 运行时 |
| GUI | Web 页面 + 内嵌 HTTP 服务器 | 不需要 Electron（太大），浏览器即 UI |
| 后台通信 | WebSocket | 双向实时，自动重连 |
| 渲染引擎 | FFmpeg + Blender | 免费，功能完整 |
| 自动更新 | GitHub Releases 检查 | 简单可靠 |
| 日志 | 文件日志 + 云端同步 | 故障可追溯 |

## 七、安全设计

| 风险 | 措施 |
|------|------|
| 未授权使用 | 授权码验证，绑定设备 |
| 命令注入 | 白名单执行路径 |
| 数据泄露 | 素材不上传云端，只传元数据 |
| 通信篡改 | WebSocket 加密（wss） |
| 客户端伪造 | 客户端签名验证 |
