# Remote PC Agent — Web 版产品方案

## 一、产品定位

一套浏览器即用的远程 PC 控制方案。运营人员打开网页登录，配置好本地隧道，即可让云端 AI 操控本地电脑执行任务。

**核心价值：零安装，打开浏览器就能用。**

## 二、整体架构

```
运营人员浏览器                    entos.top 云端
┌──────────────────┐          ┌──────────────────────┐
│  /agent/ 页面     │  HTTP   │  FastAPI 后端         │
│  ┌ 素材目录       │ ──────→ │  ┌ 任务管理           │
│  └ 开始任务       │ ←────── │  └ 隧道转发           │
└──────────────────┘          └──────┬───────────────┘
                                     │ WebSocket 隧道
                                     │
                              ┌──────┴───────────────┐
                              │ 你的 Windows 电脑     │
                              │  ┌ 隧道客户端         │
                              │  ├ Blender / FFmpeg  │
                              │  └ 素材文件           │
                              └──────────────────────┘
```

## 三、用户使用流程

### 3.1 首次使用（5 分钟配置）

**第 1 步：打开页面**
```
https://entos.top/agent/
```

**第 2 步：下载隧道客户端**
```
https://entos.top/scripts/tunnel_client.py.txt
```
保存为 `tunnel_client.py`

**第 3 步：运行客户端**
```
pip install websockets
python tunnel_client.py
```
看到 `[+] Connected` 表示连接成功。最小化即可。

**第 4 步：页面自动显示"已连接"**
配置完成，之后每次打开页面就能用。

### 3.2 日常使用

```
打开 https://entos.top/agent/
  ↓
页面显示当前连接状态
  ↓
选择素材目录（或填写路径）
  ↓
选择任务模板（足球宣传片 / 探店 / 活动集锦）
  ↓
点"开始任务"
  ↓
页面显示实时进度条 + 日志
  ↓
完成后显示输出文件路径
```

## 五、技术方案

### 5.1 云端组件

| 组件 | 技术 | 说明 |
|------|------|------|
| 前端页面 | Astro 静态页面 + 原生 JS | 部署在 entos.top/agent/ |
| 后端 API | FastAPI（已运行，端口 8765） | 新增 /api/agent/* 路由 |
| 任务数据库 | SQLite（evolution.db） | 存储任务状态和记录 |
| 隧道服务端 | WebSocket（端口 8768，已运行） | 接收本地客户端连接 |

### 5.2 本地组件

| 组件 | 文件 | 说明 |
|------|------|------|
| 隧道客户端 | tunnel_client.py（下载） | 连接阿里云 WebSocket |
| 渲染引擎 | Blender / FFmpeg（已有） | 本地执行渲染 |
| 配置 | 无 | 隧道客户端无配置，直接运行 |

### 5.3 任务流程

```
页面 POST /api/agent/task  →  创建任务记录（状态: pending）
  ↓
隧道服务端发现有新任务 → 通过 WebSocket 推送给本地客户端
  ↓
本地客户端收到任务 → 调用 Blender/FFmpeg 执行
  ↓
进度实时回传 → 存入数据库
  ↓
页面轮询 GET /api/agent/progress/{id} → 显示进度条
```

## 六、优势对比

| 维度 | APP 方案（exe） | Web 方案 |
|------|---------------|---------|
| 安装 | 下载 exe，双击安装 | ❌ 不需要安装 |
| 更新 | 重新下载安装包 | ✅ 刷新页面即更新 |
| 跨平台 | Windows 仅 | ✅ 任何设备（手机/平板/Mac） |
| 分发 | 发安装包给客户 | ✅ 发一个链接 |
| 开发成本 | 高（打包 PyInstaller） | ✅ 低（现有网站加页面） |
| 离线使用 | ✅ 可以 | ❌ 需要联网 |
| 本地资源访问 | ✅ 直接调用 | ⚠️ 需隧道转发 |
