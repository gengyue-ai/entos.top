# entos.top

企业级 AI Agent 工程实践指南。

## 这是什么

一个持续更新的知识站点，专注于企业级 AI Agent 系统的工程实践——从方法论到代码实现。

**核心体系：**
- 🎯 **Harness 方法论** — 上下文管理·验证闭环·输出约束·知识沉淀
- 🧠 **三层记忆系统** — 工作记忆·情景记忆·语义知识
- 🔄 **自进化引擎** — 从执行数据中自动发现模式→生成改进方案
- 🤖 **Loop Engineering** — 多层反馈循环的设计、观测与治理

## 内容

44 篇文章（持续增长），按知识域组织：

| 域 | 篇数 | 说明 |
|---|------|------|
| 基础理论 | 3 | ReAct、记忆系统、LLM 边界 |
| 架构设计 | 5 | 编排模式、MCP/A2A、多 Agent 协作 |

## 技术栈

- **前端：** Astro（静态站点生成）
- **部署：** Nginx / 阿里云
- **后端：** FastAPI（Demo / 评论 API）
- **数据：** SQLite
